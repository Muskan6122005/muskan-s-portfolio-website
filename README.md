# Muskan Yeshmin Ali | Interactive Cyberpunk Portfolio

Welcome to the source code for my personal interactive portfolio!

## 🚀 Live Demo
**Check out the live portfolio here:** [https://muskan-portfoli.netlify.app/](https://muskan-portfoli.netlify.app/)

## ✨ Features
- **Cyberpunk / Dark Mode Design:** A sleek, fully dark-themed UI.
- **Interactive "Crazy Cards":** Dynamic neon hover spotlights that track the mouse cursor.
- **Animated 3D Background:** A scrolling grid pattern layered behind floating, pulsing ambient light blobs.
- **Custom Mouse Cursor:** Hidden system mouse, replaced by a neon dot and an expanding trailing circle on hover.
- **Glitch Hero Effect:** The main header utilizes a complex text shadow animation mimicking digital glitches.
- **Scroll Reveal Animations:** Sections slide seamlessly onto the page as the user scrolls.

## 🛠️ Built With
- **HTML5:** Semantic structure and layouts.
- **CSS3:** Advanced stylings, grid layouts, custom properties (CSS variables), `backdrop-filter` for glassmorphism, keyframe animations, pseudo-elements.
- **Vanilla JavaScript:** Event listeners, DOM manipulation, `IntersectionObserver` for scroll reveals, and cursor-tracking coordinates.

## 📥 Local Usage
To run the portfolio locally, clone down the repository:
```bash
git clone https://github.com/Muskan6122005/muskan-s-personal-blog.git
cd muskan-s-personal-blog
```
Simply open `index.html` in your favorite web browser — no build tools or servers required!
